==========================================================================
HOW TO USE THE AUTOMATED TEST FILE
==========================================================================

NO TECHNICAL KNOWLEDGE REQUIRED!

This test file will check everything automatically and show you results
in plain English with green checkmarks (good) or red X marks (problems).

==========================================================================
STEP 1: UPLOAD THE TEST FILE
==========================================================================

1. Download the file: rts-system-test.php
2. Go to your hosting file manager (or use FTP)
3. Navigate to: /wp-content/themes/hello-elementor-child-rts-v3/
4. Upload rts-system-test.php into that folder

That's it for uploading!

==========================================================================
STEP 2: RUN THE TEST
==========================================================================

1. Make sure you're logged into WordPress as an admin
2. Visit this URL in your browser:
   
   https://reasonstostay.inkfire.co.uk/wp-content/themes/hello-elementor-child-rts-v3/rts-system-test.php

3. Wait 5-10 seconds for all tests to run
4. View the results!

==========================================================================
AFTER FIXING ISSUES
==========================================================================

1. Make the fixes suggested in the test results
2. Click the "Re-run Tests" button at the bottom of the page
3. Verify the RED tests are now GREEN

==========================================================================
WHAT IF EVERYTHING IS RED?
==========================================================================

This means the theme file hasn't been uploaded or activated yet:

1. Go to WordPress admin ‚Üí Appearance ‚Üí Themes
2. Make sure "Hello Elementor Child RTS v3" is activated
3. Check the version number says 3.2.1 or higher
4. Run the test again

==========================================================================
GETTING HELP
==========================================================================

If you have RED tests that won't turn GREEN:

1. Scroll to the bottom of the test page
2. Find "Diagnostic Data Export" section
3. Copy the JSON data
4. Send it to your developer with:
   - A screenshot of the test results
   - Which tests are failing
   - What you've tried so far

==========================================================================
SECURITY NOTE
==========================================================================

This test file requires admin access. If someone who is not an admin
tries to run it, they'll get an error message. This is for security.

After you're done testing, you can delete the test file if you want:
- Delete: /wp-content/themes/hello-elementor-child-rts-v3/rts-system-test.php

Or keep it for future testing - it's harmless and can only be accessed
by administrators.
